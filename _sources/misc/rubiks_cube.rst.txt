魔方解法
========

.. versionadded:: @2019-01-26
   创建

三阶魔方

0. 定义

    * 操作

        * 上/下：最右边一层朝上/下转动 90 度
        * 左/右：最上面一层朝左/右转动 90 度
        * 顺/逆：最前面一层顺/逆时针转动 90 度
        * 双顺/双逆：最前面两层顺/逆时针转动 90 度

    * 手法

        #. 右手手法：上左下右
        #. 左手手法：上右下左(**操作最左边一层**)

    * 除非特别说明，底面中心为白色，上面中心为黄色

#. 第一层复原

    #. 黄色中心块朝上

        * 构建围绕黄色中心块的白色十字线

    #. 依次将白色棱块按另一面的颜色和侧面中心块颜色对齐，然后转移到底面

        #. 形成白色十字线，且棱块对齐

    #. 对齐角块

        #. 选择有白色的角块，将其放置在该角块另两种颜色中心块所在面的交界处
        #. 按如下规则调整，直至第一层底和边颜色都相同

            * 如白色朝右

                * 上左下

            * 如白色朝上

                * 上左左下右 上左下

            * 如白色朝前

                * 左上右下

        #. 重复直至第一层还原

#. 第二层复原

    * 特殊情况（顶层棱块全都有黄色时）

        * 右手手法 左手手法

    * 步骤

        #. 对齐（选择不含黄色的顶层棱块，将正面颜色对齐相应的中心块）
        #. 远离（看上一步骤选择棱块的上面颜色对应左侧还是右侧的中心块，
           将上层往右或左（相反方向）转 90 度
        #. 远离相反方向手法，即 左或右手手法
        #. 魔方整体朝远离同方向转体 90 度
        #. 远离同方向手法，即右或左手手法
        #. 重复直至第二层全部还原

#. 第三层 顶面十字

    * 如果中间为黄色横线，则横放

        * 顺上左下右逆

    * 如黄色组成拐弯

        * 反L，9点方向

            * 顺 两次右手手法（上左下右）逆

        * 3 点半

            * 双顺（两层一起转）上左下右 双逆

    * 点 凹

        * 各一次

            * 顺 右手手法（上左下右） 逆 顺 两次右手手法 逆
            * 或 顺 右手手法 逆 双顺 右手手法 双逆

#. 顶面还原

    * 鱼头朝左下

        * 鱼鳍在左边

            * 下右上右下右右上

        * 鱼鳍在右边

            * 上左下左上左左下

    * 如没有鱼头，二碰四不碰（不是黄色的块）

        * 上左下左上左左下

* 还原顶层角块

    * L 公式

        * 相同颜色角块放右边，黄色朝前

            * 上上 底底 下 右 上 底底 下 左 下

        * 无相同颜色角块

            * 顺 上右下右 上左下逆 上左下右 下顺上逆

* 还原顶层棱块

    * 黄色面朝上，好的一面放后，中间的看去左还是右
    * 左

        * 上左下左上左左下 调整鱼头 下右上右下右右上

    * 右

        * 下右上右下右右上 调整鱼头 上左下左上左左下
