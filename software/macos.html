文件权限问题¶

10.14 版本在 iTerm2 下查看文件（例如 Time Machine 备份文件夹）时会出现:

ls: : Operation not permitted

而 10.14 之前的版本并没有这个问题。原因是 10.14 开始的 macOS 增加了更多的 隐私安全控制，按如下步骤配置权限即可修复问题：

  1. open /System/Library/PreferencePanes/Security.prefPane/
  2. 选择 隐私/Privacy
  3. 左下角解锁获取权限
  4. 选择 Full Disk Access ，增加相应的 App

快捷键¶

参考 Mac keyboard shortcuts

  • 截图

    • 屏幕部分画面 ⌘ + ⇧ + 4
    • 截取当前窗口 ⌘ + ⇧ + 4，SPACE
    • 全屏截图 ⌘ + ⇧ + 3

    截图文件保存在桌面，格式为 PNG。

  • 浏览器

    • ⌘ + L 光标定位到地址栏，按 TAB 取消。

  • 编辑

    • ⌘ + z Undo
    • ⌘ + ⇧ + z Redo
    • ⌘ + v Paste

  • 窗口管理

    • ⌘ + ⌥ + h 将所有已打开的程序窗口 （除了当前正在运行的应用程序窗口）隐藏到 Dock 栏上。
    • ⌘ + ⌥ 并点击桌面上空白区域，将除 Finder 窗口以外的所有窗口隐藏到 Dock 栏，直接显示桌面。
    • ⌘ + m 最小化到右下角
    • ⌘ + w 关闭窗口
    • 文件剪切： 选中目标文件，⌘ + c 复制，然后 ⌘ + ⌥ + w 移动到目标目录。

  • Finder

    • ⌘ + o 打开文件
    • 选择文件后
      • 快速预览文件
        • SPACE
        • ⌘ + y
        • ⌘ + ⌥ + y 全屏模式
      • ↩︎ 重命名（Windows 下是打开文件）
    • ⌘ + ↑ 回上一级目录
    • ⌘ + ⌥ + p 隐藏或显示路径栏
    • ⌘ + ⌥ + s 隐藏或显示边栏
    • ⌘ + / 隐藏或显示状态栏。
    • ⌥ + 双击左键 在新窗口中打开该文件/文件夹，并关闭当前窗口。

  • 多窗口应用

    • ⌘ + ` 多窗口之间切换
    • 按住 ⌘ 拖动后面窗口，此时焦点任是前窗口

  • ⌃ + ⇧ + 电源键 关闭屏幕

  • 将文件拖到终端，会直接得到其路径

  • 触控板

    • 双指左右滚动：在页面之间轻扫
    • 双指从右边缘向左轻扫：显示通知中心
    • 张开拇指和其他三指：显示桌面
    • 提拢拇指和其他三指：显示启动台（Launchpad）
    • 三指左右轻扫：在全屏幕显示的应用之间切换
    • 三指向上轻扫：调度中心
    • 辅助功能开启更多功能后
      • 在窗口标题栏处三指拖移，移动过程中可以两指

  • 声音/屏幕亮度“小调节”：

    按住 ⇧ + ⌥ 后按音量/亮度，以更细微的粒度调节
